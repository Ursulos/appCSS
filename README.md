# Projet appCSS

Ce projet est un apprentissage de HTML et CSS.  
Il contient une page de connexion stylisée "index" pour les prises de notes et un dossier `public` pour les fichiers CSS, images, pratique (pour les exercices) et webfonts pour les icones.

## Structure des dossiers

- `index.html` → page principale
- `public/css/` → fichiers CSS
- `public/images/` → fichiers images
- `public/pratique/` → fichiers exercices pour m'entrainer
- `public/webfonts/` → fichiers télécharger depuis fontawsome

## Instructions

1. Ouvrir `index.html` dans le navigateur
2. Modifier les fichiers CSS dans `public/css/`
3. Pousser les changements sur GitHub
